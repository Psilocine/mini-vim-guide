# 第三十六天

## devTools 调试

打开调试

- 打开：`command + option + i`
- 打开并 `focus` 至 `console` 面板：`command + option + j `
- 打开并开启聚焦 `dom` 元素：`shift + command + c`

`sources` 面板搜索文件：`ctrl + p`

`搜索命令`：`shift + command + p`

聚焦 `console` 面板：`` ctrl + ` ``

清空 `console`：`command + k`

在 `sources` 面板中显示隐藏底部面板：`esc`

切换面板：`command + [ / ]`
