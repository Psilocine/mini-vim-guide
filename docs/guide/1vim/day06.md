# 第六天

## 认识文本对象

### 语法

- `operator` + (`i` 内部/`a` 外部) + 文本对象
- 可视化模式 + (`i` 内部/`a` 外部) + 文本对象

### 对象

- `w` - 单词
- `(` - `()`
- `b` - `()`
- `[` - `[]`
- `{` - `{}`
- `B` - `{}`
- `<` - `<>`
- `t` - `xml` 标签
- `‘` - `‘’`
- `“` - `“”`
- `` ` `` - ` `` `
- `s` - 句子 `sentence`
- `p` - 段落 `paragraph`
- `vim-textobj-arguments` - `ia`（不包含分隔符）`aa`（包含分隔符）
- `vim-textobj-entire` - `dae` 删除当前文本所有内容 `vae` 选中当前文本所有内容 `die` `vie` 同理
