# 第六天

## 认识文本对象

### 语法

1. `operator` + (`i` 内部/`a` 外部) + 文本对象
2. 可视化模式 + (`i` 内部/`a` 外部) + 文本对象

### 对象

1. `w` - 单词
2. `(` - `()`
3. `b` - `()`
4. `[` - `[]`
5. `{` - `{}`
6. `B` - `{}`
7. `<` - `<>`
8. `t` - `xml` 标签
9. `‘` - `‘’`
10. `“` - `“”`
11. ` - ``
12. `s` - 句子 `sentence`
13. `p` - 段落 `paragraph`
14. `vim-textobj-arguments` - `ia`（不包含分隔符）`aa`（包含分隔符）
15. `vim-textobj-entire` - `dae` 删除当前文本所有内容 `vae` 选中当前文本所有内容 `die` `vie` 同理
