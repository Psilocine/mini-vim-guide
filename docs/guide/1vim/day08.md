# 第八天

## 搜索

### 单行单字搜索

1. `f` - 进入搜索，按下字符向后匹配移动
2. `F` - 同 `f`，但是反转方向
3. `t` - 进入搜索，按下字符向后匹配，但光标在匹配字符前一个字符上
4. `T` - 进入搜索，按下字符向前匹配，但光标在匹配字符后一个字符上
5. `;` - 重复上次的字符查找命令
6. `,` - 同 `;`，但是反转方向

### 全局

1. `/` - 向后查（非全字匹配）
2. `?` - 向前查（非全字匹配）
3. `n` - 向下一个匹配
4. `N` - 向上一个匹配
5. `#` - 全字匹配（当前光标，下一个）
6. `*` - 全字匹配（当前光标，上一个）
