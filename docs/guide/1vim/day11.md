# 第十一天

## 定位

`m` + 小写字母（单文件）/大写字母（多文件），每个字母都可以独立记录一个位置，平时只用一个即可。

1. 单文件
   1. 标记 `mm`
   2. 跳转 `` `m ``(精确到列)/`'m`（只能定位到行，推荐使用，`` ` ``键太远了）
2. 多文件
   1. 标记 `mM`
   2. 跳转 `` `M/'M ``

### jump to definition

1. 跳转到定义处：`gd(goto definition)`

### 跳转

1. 向前：`ctrl + i`
2. 向后：`ctrl + o`（记忆：`older`）

场景：`gd` 到其他文件后，`ctrl + o` 回到原来文件（多文件跳转
