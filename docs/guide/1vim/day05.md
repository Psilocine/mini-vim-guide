# 第五天

## 第三种模式 VISUAL（可视化）

在 `VISUAL` 模式下，可以对一个文本块的整体进行操作

### 如何进入

- 字符可视化模式（单字符） - `v`
- 行可视化模式（当前行） - `shift + v`
- 块可视化模式 - `ctrl + v`（配合 `hjkl` 选中，类似鼠标选中）

### 如何退出

- `esc`
- `ctrl + [`（`v`/`shift + v`/`ctrl + v`，再按一次也可以退出）

### 导航

- `VISUAL` 模式下切换可视区的光标位置，用于块选的时候首行或尾行遗漏情况：`o`
- `NORMAL` 模式下回到上一次选择的区域：`gv`
