# 第七天

## 单文件移动

### 滚动

1. 下滚动一屏：`ctrl + f`(重点掌握)
2. 上滚动一屏：`ctrl + b`(重点掌握)
3. 下滚动半屏：`ctrl + d`
4. 上滚动半屏：`ctrl + u`
5. 下滚动一行：`ctrl + e`
6. 上滚动一行：`ctrl + y`
7. 下滚动五行：`shift + j`
8. 上滚动五行：`shift + k`

### 当前光标所在行

1. 置于屏幕中央：`zz`
2. 置于屏幕顶部：`zt（top`
3. 置于屏幕底部：`zb（bottom`
4. 跳转文件首行：`gg`
5. 文件末行：`shift + g`

### 跳转指定行数

行数 + `gg/G`
