# 第七天

## 单文件移动

### 滚动

- 下滚动一屏：`ctrl + f`(重点掌握)
- 上滚动一屏：`ctrl + b`(重点掌握)
- 下滚动半屏：`ctrl + d`
- 上滚动半屏：`ctrl + u`
- 下滚动一行：`ctrl + e`
- 上滚动一行：`ctrl + y`
- 下滚动五行：`shift + j`
- 上滚动五行：`shift + k`

### 当前光标所在行

- 置于屏幕中央：`zz`
- 置于屏幕顶部：`zt（top`
- 置于屏幕底部：`zb（bottom`
- 跳转文件首行：`gg`
- 文件末行：`shift + g`

### 跳转指定行数

- 行数 + `gg/G`
